# Sidney Tech - Sistema T.M.O. 🏗️

Este é um assistente técnico desenvolvido para facilitar cálculos de medição e orçamentação de obras, focado em estudantes e profissionais de construção civil.

### Funcionalidades:
- **Cálculo de Reboco:** Determina a quantidade de cimento (sacos), areia (m³) e água (L) usando constantes técnicas (35.64 e 1.485).
- **Cálculo de Tijolos:** Estimativa automática com base na área da parede.
- **Gestão de Custos:** Cálculo do investimento necessário em Kwanza (Kz).
- **Exportação:** Gera orçamentos em formato .txt para partilha rápida.
